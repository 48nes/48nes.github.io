Project Description

For this project, we implemented a version of the Raft consensus algorithm for managing logs. To ensure that our database was accurate, we had to ensure that we could handle packet drops, follower failures, and leader failures.

We implemented the Raft election process and ensured that following a failure, a new leader could be elected and resume responding to clients.
