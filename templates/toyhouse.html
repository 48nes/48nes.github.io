Creating custom rethemes for toyhou.se has been a long-term passion project of mine. Below you can view an overview of what toyhou.se is and an archive of some of my best work.

Toyhou.se

Toyhou.se is a character management site used by artists and writers to organize and showcase their original characters. Toyhou.se allows all users access to HTML customization of profiles and premium users access to CSS customization. Toyhou.se uses the Bootstrap framework for all pages.
